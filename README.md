## 📌 Project Overview

The Smart Campus – College Management & Student Support System is a web-based application designed to manage academic activities, student safety, faculty evaluation, and campus resource utilization in a single unified platform.

It provides tools for students, faculty, and administrators to streamline attendance tracking, assignment evaluation, academic performance calculation, complaint management, and resource booking.

The goal is to simplify campus operations, improve transparency, and enhance student support services.

## 🚀 Key Features


- 🧠 SPI Calculator & Academic Performance Tracking
- 🛡️ Ragging & Incident Complaint Reporting System
- 📅 Campus Resource Booking System
- 💰 Scholarship & Loan Information Guidance
- 🔎 Plagiarism Detection for Assignments

- View attendance and SPI 
- Submit assignments
- Track attendance percentage
- Raise complaints (ragging, fighting, incidents)
- Access scholarship & loan information
- Book campus resources
- Practice problem-solving questions

### Faculty
- Check submitted assignments

## ⚙️ Getting Started

### Prerequisites

Ensure you have the following installed:

- Node.js 
- Git
- MongoDB 

Check installation:

```bash
node -v
git --version
📥 Installation

Clone the repository:

git clone https://github.com/Dhairy1287/CollegeProbSolving.git

Navigate to the project directory:

cd smart-campus

Install dependencies (if using Node.js):

npm install
▶️ Usage

Start the development server:
nodemon index.js
npm run dev

To start front end:
cd..
cd client
npm run dev

Open the application in your browser:

http://localhost:3000
